# Guitar-tuner

基于 Android 的吉他调音器，支持实时音高检测与偏差曲线展示。

## 功能
- 使用 Oboe/AAudio 录音，JNI 连接原生层。
- 采用 YIN（CMNDF）算法识别音高，并进行平滑与稳定判定。
- 实时展示最近 5 秒的半音偏差曲线，纵轴自动缩放。
- 支持滑条配置关键算法参数（窗口大小、平滑系数、噪声门限、YIN 阈值）。
- 参数设置移至主菜单，并分别进入“特殊调弦”和“算法参数”两个子页面。
- 图表中突出显示偏差为 0 的基准线。
- 使用重叠窗口分析，提高图表刷新频率与连续性。
- 采用频率中值滤波与更严格的 YIN 置信度筛选，提高准确度与稳定性。
- 支持自定义每根弦的目标音名（特殊调弦），在设置页配置。

## 参数说明
- 窗口大小：参与分析的采样点数。越大越稳、抗噪更好，但响应更慢、对快速变化不敏感。
- 平滑系数：频率平滑的权重（指数平滑）。越小越稳、抖动更少，但反应更迟钝；越大越灵敏但更抖。
- 噪声门限(dB)：低于该 RMS dB 时认为无信号。阈值越高越容易忽略弱音，越低越容易把噪声当成信号。
- YIN 阈值：CMNDF 的“置信”门槛。越小越严格、误检更少但可能漏检；越大更容易出结果但可能不稳定。
- 调弦设置：可为 1-6 弦分别指定目标音名，用于特殊调弦（如 Drop D 等）。

## 备注
- 识别窗口默认 16384 采样点以提升稳定度，响应会略慢。
- 需要麦克风权限。
- 偏差单位为半音。
